# Theme — HBM / Memory Super-Cycle

_Wiki · generated 2026-06-19 · cross-company theme · sources: equity calls (`../_equity_calls/Semis`), briefing roll-ups (`../_briefings/by-ticker`), earnings transcripts, company pages. Company pages: [../00_INDEX.md](../00_INDEX.md) · themes index: [00_THEMES.md](00_THEMES.md)._

## What it is / why it matters
Memory has gone from the cyclical victim of the semi complex to its tightest, most profitable corner. HBM — the stacked DRAM that feeds AI accelerators — is **sold out for all of CY26 on price and volume, including HBM4** ([MU page](../MU.md); Mehrotra, Micron Q1 FY26, 2025-12-17), and that scarcity has spread back into conventional DRAM and NAND because every HBM die displaces ~3x the wafer of a commodity DDR5 die. The result is a pricing melt-up that has run the memory names parabolic (MU +30% in a week, $748→$971; Susquehanna 2026-05-30 via [MU briefing](../_briefings/by-ticker/MU.md)) and re-rated the equity story from "trough multiple on cyclical earnings" toward an earnings-durability framework underwritten by multi-year contracts. The buy-side question this theme resolves: **is this a structurally longer up-cycle (LTAs + supply discipline + AI memory-intensity), or the textbook commodity top that always follows astronomical ASPs?** The on-disk evidence (May–Jun 2026) is two-sided and the debate is no longer direction but duration.

- **HBM4/HBM4E primer — the architectural breakpoint & the 3-supplier state (Atlas Peak Research "HBM4 and HBM4E: The Memory Wall Becomes the System Architecture Problem," 2026-04-03).** JEDEC finalized HBM4 in April 2025: HBM4 is the first generation to **double the interface to 2048-bit** (32 channels, 2 pseudo-channels/channel) rather than lean on pin-speed, and HBM4E extends the same 2048-bit base to 16 Gb/s. Value capture is moving **upstream/sideways into the logic base die, advanced packaging and substrate materials (ABF ~100% share)**, not only DRAM bits — HBM4/HBM4E are "memory-plus-logic-plus-packaging systems," so supply stays structurally tight because the bottleneck is a chain (TSV DRAM → base-die foundry → interposer/CoWoS → substrate → thermals/qual), not one fab step. - **Pricing trajectory — DDR5/LPDDR5 contract pricing already inflecting hard by 1Q26 (SemiAnalysis, 2026-02-06).** DDR5 contract pricing moved **+70% QoQ in 1Q26**, LPDDR5 **+35% QoQ**, reaching **~$14/GB (DDR5) and ~$11.5/GB (LPDDR5)**; on a YoY basis (vs 1Q25) that's **+638% DDR5, +369% LPDDR5**, with SemiAnalysis forecasting a further **~2x pricing increase from 4Q25 to 4Q26** (p.25). This is the earliest datapoint on disk for the pricing melt-up this page otherwise tracks starting ~April 2026 — the trajectory was already visible in February.
- **Divergence from the capacity-race narrative: OpenAI's ASIC runs 8-Hi HBM4, not 12/16-Hi (SemiAnalysis "Scaling the Memory Wall", 2025-08-11).** Not every accelerator design is chasing maximum HBM stack height. OpenAI's ASIC uses **8-Hi HBM4** rather than the 12-Hi/16-Hi configurations more common elsewhere — a deliberate **bandwidth-over-capacity tradeoff** enabled by high-rank expert parallelism in its model architecture (p.19-20). Worth flagging as a live counter-thread to this page's capacity-build-out framing: not every buyer is bidding for more HBM capacity per accelerator — some are optimizing for bandwidth density instead, which if it spreads would blunt one leg of the HBM-shortage bull case (fewer GB/accelerator demanded even as unit shipments grow).
- **China-domestic-HBM wildcard: CXMT ramping HBM2 8-Hi, Huawei's memory affiliates still entity-listed (SemiAnalysis "Scaling the Memory Wall", 2025-08-11).** CXMT reached **HBM2 8-Hi mass production in H1 2025**, with TSV capacity guided to match Micron's by year-end (p.10-12) — a slower, lower-tier ramp than the CXMT-IPO/CXMT-sold-out datapoints tracked elsewhere on this page (which concern conventional DRAM, not HBM). Huawei's **XMC + SJSemi HBM-affiliated entities remain under US entity-list restrictions** (p.10-12). No direct wiki ticker currently captures this; flagging as a watch-item wildcard on the domestic-China side of the shortage debate, distinct from and behind the CXMT-conventional-DRAM threat already covered above.
- **HBM packaging/technical architecture layer — complements the supply/demand numbers above (SemiAnalysis "Scaling the Memory Wall", 2025-08-11).** MR-MUF (SK Hynix's mass-reflow molded-underfill) vs NCF/TCB (non-conductive-film thermo-compression bonding, Samsung/Micron) packaging approaches are compared across the three memory makers (p.6-9) — the process-technology backdrop to the MR-MUF-vs-hybrid-bonding debate tracked on [cowos-packaging.md](cowos-packaging.md). On the base-die supply chain: **TSMC fabricates HBM4 base-die wafers (N12/N3) for Nvidia/Broadcom/hyperscaler designers**, since the memory makers themselves mostly lack leading-edge logic capability (p.25-26, 40-42) — full mechanics of this on [cowos-packaging.md](cowos-packaging.md). SemiAnalysis's framing on who actually captures HBM economics: **Nvidia is "arguably the most profitable HBM vendor in the world"** by capturing GPU-level margin on top of HBM content it resells inside the accelerator (p.42-43) — a useful counter to any read of HBM profit pools that stops at the memory maker.

## Key debates
- **New bear branch — efficiency/substitution:** the cycle can remain tight while accelerator architectures reduce HBM intensity per token. The observable test is whether QCOM can validate its 4–8× decode/TCO claim at application level and secure LPDDR allocation; until then HBC is an execution-dependent architecture, not a change to the canonical HBM supply/pricing ledger (Citrini Semis, 2026-07-20).
- **Bull / where the money is — duration, not direction.** The cleanest bull is UBS Arcuri on MU: PT $535→$1,625, "path to >$100 EPS," C27/28/29E $155/$167/$117, LTAs locking ~30% of industry DDR bits and ~$400bn FCF C27-29E ([MU briefing](../_briefings/by-ticker/MU.md), 2026-05-26/31 — flagged "most controversial UBS note recently"). Supporting voices: DB (Weathers, Buy) "DRAM tightness extends to 2028+" (2026-06-17); GS (Schneider) "ongoing tightness through CY27, significant upside to Street" (2026-06-09); Citi 8x C27 EPS from 5x (2026-05-19); Susquehanna $1,750 (2026-05-30). Mechanism: HBM's ~3:1 die displacement + node-driven bit decline + multi-year cleanroom/EUV lead times keep supply short while LTAs/SCAs convert the equity into an earnings-durability story (JPM, 2026-05-28). Suppliers are now optimizing for **durability over sequential momentum** — most will be at ~80% gross margin by year-end and are "securing the durability of earnings rather than seeking additional sequential upside" (JPM/Jay, 2026-05-28). GS (Junie Yoon) makes the cleanest "longevity is underestimated" case: DRAM names at 5-6x earnings disbelieve durability, but a 9x earnings multiple is "conservative" vs semis peers with similar margins/moats at high-teens-to-20x — and the bear-case sensitivity (DRAM ASP **−30% in 2027 then −30%/−50% in 2028 still leaves DRAM GM ~40%**) is the downside protection that justifies an earnings (not P/B) multiple; lower crash-risk than past cycles because hyperscaler demand visibility lets suppliers spend capex efficiently rather than overbuild (GS "memory", 2026-06-08). UBS Arcuri's incremental tell into the Micron print: the Street is fixated on DDR, but **HBM pricing is "what is really surprising to the upside,"** and the **LTAs (not pricing) are the re-rate catalyst** — investor demand is "rabid" (40+ call requests/day, "panicked about missing these moves," 4 death threats after the PT-to-$1,600 note went viral) (UBS "Arcuri call", 2026-06-15).
- **Supply discipline is the load-bearing assumption.** Capex has *not* yet ramped meaningfully because it takes 12-18 months to source tools and find fab space; the industry mis-forecast a mid-26/early-27 *down*-cycle and is only now turning the crank (Micron ex-VP, [procurement call](../_equity_calls/Semis/2026-06-10_Micron-exVP_procurement.md), 2026-06-10). The 2028 governor is EUV: ASML's ~90 tools by end-CY27 are insufficient once Micron/Samsung/Hynix each add ≥100k wafers/month — Samsung/Hynix need ~20-25 tools each, Micron ~10-15, TSMC already ~40 (JPM/Jay, 2026-05-28). DRAM↔HBM mix is fungible overnight (no equipment change), so makers keep starving commodity DRAM to feed HBM — the "die penalty" that sustains the shortage (JPM, 2026-05-28; ex-VP, 2026-06-10). UBS (Arcuri) reframes the EUV governor more benignly — ASML isn't a physical bottleneck until ~**110 EUV slots (~$225B WFE)**, beyond which it must expand floor space, and it's already getting the same 8-10-quarter visibility as US toolmakers, so supply "is not going to be the limitation"; UBS raised 2028 WFE to $250B and floats 2029 into the $300B range, with memory not entering sustained oversupply until **~mid-2028 at the earliest** (UBS "Arcuri call", 2026-06-15). MS semicap (Shane) sizes WFE at **$149B (2026) → $191B (2027, +28%) → mid-teens growth (2028)**, flags 2026 DRAM WFE revisions "even more aggressive" (~+60% to ~$50B), but underscores remarkable **NAND capital discipline** — a NAND-WFE "resurrection" is a 2027 story, not a 2026 pull-in (MS "memory & semicap", 2026-06-22).
- **Bear / what could break it — the commodity snap-back.** The sharpest bear on disk is the **ex-Micron VP of Procurement** (2026-06-10): the market is pricing peak earnings as sustainable — "a highly catastrophic assumption." HBM differentiation is *temporary* (once a 2nd/3rd vendor qualifies at each customer "it already has become a commodity"); Micron's cost structure is worse than Samsung's and ~level with SK Hynix; and "your $900 stock right now will become a $50 stock again," possibly normalizing "within this calendar year or early next" as Micron/SK/Samsung bring disruptive new-fab capacity online ~a year out. He frames the AI buildout as a multi-layer bull-whip ("beer game") removed from end demand that can't amortize ~$6-700B of infra spend. The XP GPU specialist independently calls HBM "comoditizado" — once multiple vendors are NVIDIA-qualified the timing advantage decays to a commodity ([XP/GPU-memory](../_equity_calls/Semis/2026-05-27_XP_GPU-memory.md), 2026-05-27).
- **Anthropic implied to be building an in-house ASIC (Avril, 2026-06-23).** TrendForce reads the Anthropic–Micron deal — **server DRAM + enterprise SSD in 2027 plus HBM in 2028** — as direct evidence that **Anthropic is developing its own in-house ASIC** (you only need HBM if you have your own accelerator); Anthropic is pursuing the **same combined deal with SK Hynix and Samsung** (StoneX/TrendForce "Memory expert call — Avril", 2026-06-23). On the China supply side, both sources play down the near-term threat: **CXMT is sold out, with its +20K end-2026 increase mostly an HBM *attempt*** that won't ramp in volume (technology too old; output only good for low-grade consumer LPDDR5/DDR5), and Conor hears a "good chunk" of CXMT capacity is earmarked for **HBM inside China** for the domestic AI ramp — not a US oversupply threat (Redburn "Conor O'Mara Asia Trip Takeaways", 2026-06-23; StoneX/TrendForce, 2026-06-23). NAND: global wafer capacity *declines 3% in 2026* as Samsung/SK Hynix convert NAND to the more-lucrative DRAM (+6% 2027, +11% 2028); **YMTC is the more concerning Chinese supplier** (uses domestic tools to expand, strong in consumer-grade QLC) but its **QLC is only ~13% of output**, capping it to the consumer segment vs enterprise-SSD-critical Kioxia/Solidigm (StoneX/TrendForce, 2026-06-23).
- **Second-source qualification & China.** Customers force-qualify Chinese DRAM (CXMT) on cheap LPDDR Micron/Samsung/Hynix won't supply at current ASPs — "once they're in, they're in," a lasting share leak at the low end (ex-VP, 2026-06-10). CXMT filed its IPO at ~50% net margin, ASP +~80-85% q/q, but can't hit server or Apple grade; bottleneck is cleanroom + equipment, not policy — and US discussion of banning all equipment sales to China could cap it (JPM/Jay, 2026-05-28). HBM4E adds a wildcard: customer-supplied custom logic base die would *de*-commoditize HBM — but the ex-VP doubts it scales on warranty/yield/liability grounds (2026-06-10). Redburn's bus tour corroborates the *de*-commoditization vector: HBM is moving to **increasing base-die customization** to deliver a system-level (not commodity-chip) solution — "new agreements bind the financial and technological fortunes of the AI supply chain more closely together," suppressing double-ordering and supporting through-cycle margins (Redburn, 2026-06-17).
- **DRAM > NAND on the supply-risk axis (MS/Bernstein).** The buy-side preference has swung toward NAND/SanDisk, but MS argues that's backwards on a 2-3yr view: fundamentals are **"10 out of 10 for DRAM, ~8/10 for NAND,"** and some NAND **2028 LTA lock-in prices are *down* vs 2027** (materially so for enterprise TLC) — softened by built-in escalators — whereas DRAM long-term negotiations "go very well"; MS prefers Micron/DRAM and reads DRAM as the primary AI bottleneck (MS "memory & semicap", 2026-06-22). The NAND↔HDD substitution math (Bernstein expert, ex-WDC NAND VP): the TCO crossover where NAND takes HDD share sits at **~1.5-3x $/GB** (vs the prior ~5x); the gap blew out to ~15-25x in the run-up (vs ~4-5x in Aug-2025), so the swing back toward HDD is "somewhat inevitable" as builders re-tier away from the AI "gold rush" all-flash sloppiness. NAND's peakiness is structural: AI needs the **latest NAND node** (only ~30-35% of capacity), and the under-invested NAND base can't lift that fast — but the new LTAs are only ~25% financially guaranteed (e.g. SanDisk's $11B on $42B), so a −50% market price translates to ~−25% effective ASP, and "tearing up" means renegotiating for more years, not walking (Bernstein "HDD/NAND", 2026-05-04).
- **New structural risk to the HBM *market size* (Redburn): W2W bonding + optically-networked DRAM.** Redburn's BESI work surfaces two technologies that could shrink the HBM TAM itself, not just commoditize it. **Wafer-to-Wafer (W2W) hybrid bonding** (imec/EVG showed 200nm W2W at sub-40nm overlay) could bond DRAM *pairs* before dicing/KGD-sort, halving the per-stack bond count — though yield penalties on ~95% leading-edge DRAM yields limit it to ~2-high. More radically, **optically-networked commodity DRAM** (a large LPDDR pool reached via optics, 3-4x HBM pin speed) could "reduce the size of the HBM market altogether" — but ~300ns light↔electrical conversion latency keeps HBM on the critical path, so Redburn judges it a long-dated revolution, with HBM staying for low-latency delivery and optical DRAM as a prefetch tier (Redburn "BESI", 2026-06-15). Net: another reason the bull case rests on HBM *durability* assumptions.
- **CPU/server-DRAM demand leg quantified (UBS).** UBS's server-CPU TAM work underwrites the DRAM bit-demand uplift: server-CPU units 23M (2025) → **63-70M by 2030 (30% CAGR)** on agentic AI, with AI-CPU (head-node + standalone) climbing 3M → 33M units; standalone CPUs outside the rack go 0 → 13M — each pulling LPDDR5X SO-DIMM/SoCAM content. AI-CPU ASPs rise on core count ($2,420 → $3,769) (UBS "CPU TAM", 2026-05-21). Consistent with JPM's "AI-CPU could lift server DRAM bit demand >15-20%."
- **Timeline / inflection.** Near-term catalyst is Micron's F3Q26 print (June 24): Street rev ~$34.7B vs guide $33.5B, FY27/28 EPS pegged ~$111/$116 ([MU briefing](../_briefings/by-ticker/MU.md), 2026-06-12). The structural inflection both camps point to is **~mid-2027 to 2028**, when new fabs land — the bull says EUV scarcity caps that supply; the bear says it breaks the price. UBS converges on this: sustained memory overcapacity "not until the middle of 2028 at the earliest," so WFE wouldn't roll until ~2029 (UBS "Arcuri call", 2026-06-15). The cleanest single read across the complex now (MS): DRAM fundamentals 10/10, NAND 8/10, with the buy-side so crowded ("every hand in the room owns memory") that dips get bought — and Apple's Tim Cook calling memory pricing a "100-year flood" (WSJ) is the demand-side exclamation point (MS "memory & semicap", 2026-06-22).

- **SK Hynix's record US listing + HBM 2027 price-doubling (2026-07-10).** SK Hynix priced its Nasdaq ADR at **$149, raising ~US$26.5bn — the largest-ever first-time share sale by a foreign issuer on US markets** (past Alibaba), 7x oversubscribed with anchor allocations cut on demand, ~3% ADR premium; proceeds earmarked for DRAM/HBM capacity + ASML EUV (Reuters/Bloomberg/FT via Twitter corpus; DIGITIMES, 2026-07-10). DIGITIMES: **HBM prices set to more than double in 2027** as Rubin drives demand and supply deals tighten — but "HBM customers will wait until 2028 for the capacity" the raise funds. Competitive watch: Bloomberg profiled **China's CXMT** building a top-to-bottom memory supply chain to challenge SK Hynix/Samsung/Micron (structural low-end threat, long-dated). See [SKHYNIX](../SKHYNIX.md), [MU](../MU.md).
